# 기능 정의

1. 사용자
   1. 로그인
      1. 이름, 비밀번호, 이메일
      2. 페이스북, 구글 등 타계정 로그인 기능
   2. 스탬프 적립 기능
      1. QR코드 스캔으로 권한
   3. 메인화면
      1. 스탬프 개수
      2. 스탬프 사용 내역
      3. 보유 쿠폰
      4. 소멸 예정 쿠폰
   4. 쿠폰 관련 기능
      1. 쿠폰 사러 가기
      2. 쿠폰 설명 확인
      3. 쿠폰 교환
      4. 쿠폰 사용
   5. 마이페이지
      1. 사용자 정보 수정
2. 관리자
   1. 사용자 QR코드 통해 스팀프 인증
   2. 관리자 정보
